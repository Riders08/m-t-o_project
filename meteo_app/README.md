# m-t-o_project
Réalisation d'un application météo avec API externe avec Flutter

# Lancement du projet 

  $ cd meteo_app
  $ flutter run

# Nettoyer le projet 
  $ flutter clean

# Relancer le projet avec nouvelles importation
  $ flutter get pub

# Generer les fichiers de language
  $ flutter pub run intl_utils:generate
